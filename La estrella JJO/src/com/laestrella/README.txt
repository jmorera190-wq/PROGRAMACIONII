Proyecto Java (IntelliJ) - La Estrella (MVC, Swing, In-Memory)

Instrucciones rápidas:

1. Descargar y descomprimir el archivo ZIP en tu equipo.
2. Abrir IntelliJ IDEA -> Open -> seleccionar la carpeta descomprimida.
3. Asegúrate de tener JDK instalado (Java 8 o superior).
4. Ejecuta la clase Main (src/com/laestrella/Main.java).
5. La app usará datos en memoria (no necesita base de datos). Los datos se pierden al cerrar la app.
6. Usa la pestaña "Registro" para agregar servicios y la pestaña "Consultas" para ver reportes.

Si quieres que guarde los datos en un archivo (JSON/CSV) entre ejecuciones, dímelo y lo agrego.
